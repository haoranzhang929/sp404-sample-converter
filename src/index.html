How to Use the SP-404 Sample Converter

This application helps you convert WAV files to the format supported by the SP-404 MK2 sampler. Follow these steps to use the app:

  1. Click the "Select Directory" button to choose the folder containing the WAV files you want to convert.
  2. After selecting a directory, click the "Start Processing" button to begin the conversion process.
  3. The app will process all WAV files in the selected directory and its subdirectories, converting them to 16-bit, 48kHz format.
  4. Monitor the progress and any errors in the output area at the bottom of the page.

If you encounter any issues or need further assistance, please refer to the troubleshooting section in the README file or submit an issue.

Warning!

This app will replace existing samples. Please backup your samples before processing.
